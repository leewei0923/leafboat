常量

C++ 定义常量两种方式

  1. #define 宏常量 #define 常量名 常量值
  2. const 修饰的变量 const 数据类型 常量名 常量值

sizeof关键字

实型

  1. 单精度 float
  2. 双精度 double
数据类型 占用空间 有效数字范围
float 4字节 7位有效数字
double 8字节 15-16位有效数字

字符型

作用: 字符型变量用于显示单个字符

语法: char ch = ‘a’;

注意

  1. 在显示字符型变量时, 用单引号符号括起来 , 不能用双引号
  2. 单引号只能由一个字符 , 不可以是字符串

C和C++ 中字符型变量只占用1个字符

强制转换类型前面加(int) – 转为整数型

转义字符

转义字符 含义 ASCII码值
\a 换行 , 将当前位置移到下一行头 010
\\ 表示反斜杠 092
\t 水平制表符(跳到下一个TAB位置) 009

字符串型

C语言:

char 变量名[] = "字符串值"

C++风格:

string 变量名 = "字符串值"
