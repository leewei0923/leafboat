隐式类型转换

  • 字符串与数值相加时, 数值被转换为字符串
  • 字符串参与飞假发数学运算时, 字符串被转换为数值
  • 布尔值与数值进行运算时, true视为1 , false 视为0
  • 布尔值与字符串进行相加时, 布尔值视为字符串

== 与 ====

  • 使用 == 操作符比较时, 会触发隐式类型转换
  • 值用 === 操作符比较时, 不会触发隐式类型转换
  • 一般原则出来与null比较外,尽量使用 ===

NULL和 Undefined

Null和Undefined 是javascript 中的两种原始类型

它们分别只有一个值

  • Null 的值是null
  • Undefined 的值是undefined
  • 在非严格比较下, null == undefined
