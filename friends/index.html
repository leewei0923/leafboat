怎么申请友链？

想要交换友链的小伙伴们，欢迎去本站的 Github 仓库提交一个 PR。审核通过后，就可以在这里展示啦。

请在/src/content/friends/目录下添加一个<short-name>.yaml文件，参考格式：

yml
title: 网站名称
description: 一句话介绍下你的网站或者你自己
link: 网站地址
avatar: 头像地址
