# 二、偿债能力分析的目的

# （一）了解企业的财务状况

# （二）揭示企业所承担的财务风程度

# （三）预测企业筹资前景

# （四）为企业进行各种理财活动提供重要参考

# （一）短期偿债能力分析

# （二）长期偿债能力分析

# 第二节 短期偿债能力分析

# 一、影响短期偿债能力的因素

# （一）企业内部因素

  1. 企业的资产结构，特别是流动资产

  2. 流动负债的结构

  3. 企业的融资能力

  4. 企业的经营现金流量水平

# （二） 企业外部因素

  1. 宏观经济形势

  2. 证券市场的发育与完善程度

  3. 银行的信贷政策

# 二、短期偿债能力指标的计算与分析

# （一）营运资金

# （二）流动比率

# （三）速动比率

# （四）现金比率

# （五）现金流量比率

# （六）企业支付能力系数

# 第三节 长期偿债能力分析

# 一、影响长期偿债能力的因素

# （一）企业的资本结构，即企业自有资金与负债的比例结构

# （二）长期资产的规模与结构

# （三）企业的盈利能力

# （四）企业经营现金流量

# 二、长期偿债能力的指标计算与分析

# （一）资产规模对长期偿债能力影响的分析

  1. 资产负债率

  2. 净资产负债率

  3. 股东权益比率和权益乘数

  4. 长期负债率

# （二）盈利能力对长期偿债能力影响的分析

  1. 利息保障倍数

  2. 债务本息偿付保障倍数

  3. 固定费用保障倍数

# （三）现金流量对长期偿债能力影响的分析

  1. 现金流量利息保障倍数

  2. 现金净流量全部债务比率
