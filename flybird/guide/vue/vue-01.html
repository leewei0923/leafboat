# vue 指令(v-if和v-show)

# v-if
 - v-if 指令将根据表达式 的值(true 或 false )来决定是否插入 p 元素。
 - 是否创建节点,

# v-show
 - 是否显示节点

	<div id="app">
			<div v-show="isShow">动态显示和隐藏</div>
			<div v-if="isCreate">动态显示和隐藏</div>
		</div>
		<script>
			new Vue({
				el:'#app',
					
				data:{
					isShow:true,
					isCreate:false
				}
			});
		</script>

# vue-class 绑定

# 
 - :, '冒号'绑定,动态绑定
 - 可以在class放置一个对象,然后通过在对象中控制true或者false
   就可以控制属性了
   - 对象写法的缺点:
    - 后期添加的对象的属性,并不会添加到对象中,不会被get和set拦截
   - 如果采用数组写法,就不会出现上面不会被拦截的问题

<div id="app">
			<div style="background-color: yellow;">11111111</div>
			<button @click="handleClick()">切换</button>
			<div :class="isActive? 'red' : 'yellow'">我是动态绑定class -三目写法</div>
			<div :class="classObj">我是动态绑定class -三目写法</div>
		</div>
		<script>
			new Vue({
				el:'#app',
					
				data:{
					isActive:true,
					classObj:{
						a:true,
						b:false
						//a, b ,class 名字
					},
					classArr:['a','b'];
				},
				methods:{
					handleClick(){
						this.isActive = !this.isActive;
					}
				}
			});
		</script>

# vue 条件渲染

# v-if
 - 判断传来的数据为真,显示该数据

# v-else
 - 判断传来的数据为假,显示该数据

# v-for
 - 渲染列表的命令
# 
 - 在模板中也可以使用which等表达式,v-if,v-else-if


<div id="app">
			
			<button @click="handleClick()">切换</button>
			<div v-if="isCreated">动态创建和删除--真</div>
			<div v-else="isCreated">动态创建和删除--假</div>
			<ul v-if="isCreated">
				<li v-for="data in datalist">
					{{data}}
				</li>
			</ul>
			
			<div v-else="isCreated">
				空空如也
			</div>
		</div>
		<script>
			new Vue({
				el:'#app',
					
				data:{
					isCreated:false,
					datalist:''
					
					
				},
				methods:{
					handleClick(){
						this.isCreated = !this.isCreated;
						this.datalist = ["小米","华为","oppo","vivo"];
					}
				}
			});
		</script>

# 常用内置指令
 - v-text: 更新匀速的textContent
 - v-html: 更新元素的innerHTML
 - v-if: 如果为true,当前标签才会输出到页面
 - v-else: 如果为False,当前标签才会输出到页面
 - v-show: 通过控制display样式来控制显示/隐藏
 - v-for: 遍历数组/对象
 - v-on: 绑定事件监听,一遍简写为@
 - v-bind: 强制绑定解析表达式,可以省略v-bing简写为 :
 - v-modle: 双向数据绑定
 - ref: 指定唯一表示,vue对象通过$refs属性访问这个元素对象
 - v-cloak: 防止闪现 ,与css配合: [v-cloak]{display:none}
  1. 注册全局指令
Vue.directive('my-directive',function(el ,binding){
	el.innerHTML = binding.value.toupperCase()
})
  1. 注册局部指令
directives: {
	'mu-firective': {
		bind(el , binding) {
			el.innerHTML = binding-value.toupperCase()
		}
	}
}
  1. 使用命令

v-my-directive = 'xxx'
