# 计算机属性基本使用

# 计算属性
 - 在computed属性对象中定义计算属性的方法
 - 在页面中使用 来显示计算结果
# 监视属性
 - 通过vm对象的$watch()或watch配置来监视指定的属性
 - 当属性变化时,回调函数自动调用,在函数内部进行计算
# 计算属性高级
 - 通过getter/setter实现对属性数据的显示和监视
 - 计算属性存在缓存,多次读取只执行一次getter计算
