# 变量和标识符

# 
 - Python中使用变量不需要声明,不允许使用未经过赋值的变量
 - 如果使用未经过赋值的的变量会报错
 - python是一个动态类型的语言,可以为变量赋任意类型的值,也可以任意修改变量的值

# 数值

 # 
  - 在python中分成了三种:整数, 浮点数, 复数
  - 在python中所有的整数都是int类型
  - 其他进制:二进制 0b 开头
  - 八进制 0o 开头
  - 十六进制 0x 开头

# 字符串

 # 
  - 使用三重引号可以换行,并且保留字符串的格式 ''' ''' ,""" """
 
 # 转义字符 \
  - 例子: \' 表示'
          \" 表示"
          \t 表示制表符
          \n 表示换行符  
          \uxx 直接表示unicode的编码

# 格式化字符串

 #
  - 将两个字符串在一起相加减,会拼接字符串
  - 字符串只能和字符串进行拼接,不能和其他类型的数值进行加减
  - 在创建字符串时,可以在字符串中指定占位符 %s 
 # %s
  - xxx %s %('') 
  - 表示任意字符串
  - b = 'hello %s '%'伟伟'
  - 站位符按照顺序进行填列
  - %3s 
   - 限制占位符输出的位置
  - # %3.5s
   - 字符串的长度限制在3-5之间
 # %f
  - 浮点数的占位符
 # %d
  - 数字占位符
 # 格式化字符串,可以通过在字符串前添加一个f来创建一个格式化字符串
  - c = f'hello'
  - 在格式化字符串中,可以直接嵌入变量
  - c = f'hello {a} {b}'

# 类型检查

 # 数据类型
  - 数据类型指的就是变量值的类型,也就是可以为变量赋哪些值
   - 数值:
    - 整型
     - 布尔值
    - 浮点型
    - 复数
    - 字符串
    - 空值
 # 类型检查
  - 可以检查只能是值(变量)的类型
  - type()函数用来检查值的类型
   - 该函数会将检查的结果作为返回值返回
   - 可以通过变量来接收函数的返回值

# 对象的结构

 # 每个对象中  
  - id(标识)
   - 显示唯一性
   - 可以通过id()函数来查看对象id
   - id是由解析器生成的,id就是内存地址

  - type
   - 标识类型
   - 例如: int, str, float, bool
   - 类型决定对象具有哪些功能
   - Python是一个强类型的语言,对象一旦创建,无法更改
   - 通过type()来查看类型

  - value(值)
   - 有些值是可以改变的

# 逻辑运算符

 # 逻辑运算符主要用来一些逻辑判断
  - not 可以对符号右侧的值进行非运算
  - and 逻辑与
  - or 逻辑或

# 条件运算符

 # 语法: 语句1 if 条件表达式 else 语句
  - 执行流程:
    - 条件运算符在执行时,会先对条件表达式进行求值判断
    - 如果判断结果为True,则执行语句1,并返回执行结果
    - 如果判断结果为False,则执行语句2,并返回结果


a = 90
b = 40
c = 60
max = a if a > b and a > c else b if b > c else c

print(max)

# 条件判断语句(if语句)

 # 语法: 
  - if 条件表达式 : 语句
  - 执行的流程: if语句在执行时,会对条件表达式进行求值判断
  - 如果为True , 则执行if后的语句
  - 如果为False , 则不执行
  - 默认情况下,if只会控制紧随其后的那条语句
  - 如果需要所有条件同时满足,则使用and
  - 如果需要有一个条件满足,则使用or

if False : print('你猜我会出来吗')

# python 列表脚本操作符

列表对+和 * 的操作符与字符串相似。+ 用于组合列表， * 号用于重复列表

Python表达式 结果 描述
len([1,2,3]) 3 长度
[1,2,3] + [4,5,6] [1,2,3,4,5,6] 组合
['Hi!'*4] ['Hi!','Hi!','Hi!','Hi!'] 重复
3 in [1,2,3 True 元素是否存在列表之中
for x in [1,2,3] : print(x,end="") 1,2,3 迭代

# Python 包含以下方法:

序号 方法
1 列表名.append() 在列表末尾添加新的对象
2 列表名.conut() 统计某个元素在列表出现的次数
3 列表名.extend(seq) 在列表末尾一次性追加另一个序列中的多个值(用新列表扩展原来的列表
4 列表名.index() 从列表中找出某个值第一个匹配项的索引位置
5 列表名.insert(位置的索引,对象)
6 列表名.pop([index=-1])移除列表中的一个元素(默认最后一个元素),并且返回该元素的值
7 列表名.remove()移除列表中某个值的第一个匹配项
8 列表名.reverse()反向列表中元素
9 列表名.sort(key = None, reverse =False)对元列表进行排序
10 列表名.clear()清空列表
11 列表名.copy()复制列表
