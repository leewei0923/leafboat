# jquery过滤选择器

名称 用法 描述
:eq(index) $('li:eq(2)'.css()) 获取到li元素,选择索引号为2的元素,索引号index从0开始
:odd $('li:odd').css 获取到的li元素中,选择索引号为奇数的元素
:even $('li:even').css 获取到的li元素中,选择索引号为偶数的元素

# jQuery筛选选择器(方法)

筛选选择器的功能与过滤选择器有点类似

名称 用法 描述
children(slector) $('ul').children('li') 相当于子代选择器
find() $('').find('li) 相当于后代选择器
siblings() $('').siblings('li) 查找兄弟节点,不包括自己本身
parent() $('').parent('li) 查找父代
eq(index) $('').eq() 相当于index从0开始
next() $('').next() 找下一个兄弟
pre() $('').pre() 找上一个兄弟

# 阻止冒泡

event.stopPropagation();
  1. 通过返回false来取消默认的行为并阻止事件起泡。
JavaScript
$(element).click(function(){
	return false;
});
  1. 通过使用preventDefault()方法取消默认行为。
JavaScript
$(element).click(function(e){
	e.preventDefault();
});
  1. 通过使用stopPropagation()方法阻止事件起泡。
JavaScript
$(element).click(function(e){
	e.stopPropagation();
});

# jQuery添加节点的几种方式

  1. append() 方法 父元素.appound(子元素) ,作为最后一个子元素添加

  2. prepend(); 父元素.prepend(子元素) ,作为第一个元素被添加进去

  3. before() 父元素A.before(子元素B) 把元素B插到A元素的前面,添加兄弟元素

  4. after() 父元素A.before(子元素B) 把元素B插到A元素的后面,添加兄弟元素

  5. appendTo()

子元素.appendTo(父元素) 把子元素作为父元素的最后一个子元素添加

  1. empty()

清空元素,还清空元素的事件

  1. remove() 移除元素

# width和height方法

设置或者获取宽高,不包括内边距,边框和外边距

$().width() $().height()

 #
  - innerWidth() //方法返回元素的宽度/高度(包括内边距)
  - outerWidth()/outerHeight 方法返回元素的宽度/高度(包括内边距和边框)

 # 获取可视区宽度
  - $(window).width()
  - $(window).height()
