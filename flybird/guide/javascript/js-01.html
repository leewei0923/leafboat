# javascript学习01

# 强制类型转换String

-指将一个数据类型强制转换为其他数据类型
-类型转换主要指,将其他数据类型转换为 String Number Boolean

方式一:

-调用被转换数据类型的toString()方法.
-该方法不会影响原变量
-局限:空值(null)和未定义(undefined)值无法转换
      只适用String

方式二

-调用String()函数,并将被转换的数据作为参数传递给函数
-使用String()函数强制类型转换时
  对于Number和Boolean实际上就是调用的同toString()方法
但是对于null,undefined 就不会调用toString()方法,他会将null 直接转换为"null",将undefined直接转换为"undefined"

//将其他数据类型转换为String

var a;
console.log(typeof a);
console.log(a);
//方法一
//调用a的toString()方法
//调用xx是我yy()的方法
a = a.toString();

//方式二
a = 123;
//调用String()函数,来将a转换为字符
a = String(a);

b = null;
b = String(b);

# 将其他数据类型,强制转换为 Number

转换方式一:

-使Number()函数将a转换为Numberl类型 -字符串 ---> 数字

  1. 如果是纯数字的字符串,则直接将其转换为数字
  2. 如果字符串为非数字的内容,则转换为NaN
  3. 如果字符串是一个空串或者是一个全是空格的字符

-boolean --> 数字

  1. true转成1
  2. false转成0

-Null --->数字0

-undefined --> 数字0

转换方式二

-这种方式专门用来对付我字符串

-paseInt()把一个字符串转换成一个整数

-parseFloat()把一个字符转换为一个浮点数


var a ="123";

console.log(tyoeof a);
console.log(a);
a = Number(a);

//方式二:
a = "123px";
//调用parseInt()可以把一个字符串中的有效的整数内容去出来,然后再转换为Number
a = parseInt(a)
console.log(a)
//调用parseFloat()作用和parseInt()类似,不同的是它可以获取有效的小数点

# 转为其他进制的数字

在js中,如果需要表示16进制的数字,则需要0x开头

如果需要表示8进制的数字,则需要以0开头

如果需要表示二进制的数字,需要以0b开头,兼容不太好

//十六进制
a = 0x10;
a = 0xff;
a = 0xcaf6;
console.log(a);
//八进制
//像"070"这种字符,有些浏览器会当成8进制进行解析,有些会当成10进制
b = 070;
console.log(b);

//二进制(用的比较少)

c = 0b10;
console.log(c);

//可以在parseInt(a,10)传递一个第二个参数,来指定数字的进制

a = parseInt(a,10);

# 自增和自减

自增++:

-通过自增可以使变量在自身的基础上增加1

-对于一个变量自增以后,原来的变量的值会立即增加1

-自增分成两种:后(++),前(++),无论是a++,还是++a,都会立即使原来的变量自增1不同的是a++,和++a的值不同 a++的值等于原变量的值,(自增前的值),++a的值等于原变量新值(自增后的值)


var a = 1;
a++;//使a自增加1

------------------

var c = 10;
//第一次c++,是在10的基础上自增
c++;
//第二次c++,是在11的基础上自增
c++;

# js中的变量都是保存到栈内存中的
基本数据类型的值直接在栈内存中存储,
值与值之间是独立存在,修改一个变量不会影响其他的变量

对象是保存到堆内存中的,没创建一个新的对象,就会在堆内存中开辟一个新的空间
而变量保存的是对象的内存地址(对象的引用),如果两个变量保存的是同一个队象引用当一个变量通过一个变量修改属性时,另一个也会受到影响

var obj = new Object();
obj.name = "孙悟空";

var obj2 = obj;
//修改obj的name属性
obj.name = "猪八戒";

console.log(obj.name);//猪八戒
console.log(obj2.name);//猪八戒
